# Build-Bits Interactive Demo 🤖🧩

An interactive web prototype of the **Build-Bits** social educational robot, designed for children with ADHD. The system improves attention, working memory, and executive functions through low-stimulation, adaptive block-building activities.

---

## 🌟 Key Features

* **Four Play Modes**

  * *Guided*: step-by-step support
  * *Creative*: open-ended building
  * *Memory*: reconstruct from memory
  * *Challenge*: time-based gameplay

* **Flow Zone Adaptive System**
  Dynamically adjusts difficulty based on mistakes, inactivity, and performance.

* **ADHD-Friendly Design**
  Minimal UI, soft colors, and clear interactions to reduce cognitive load.

* **Micro-Interactions**
  Smooth drag-and-drop with physical feedback (snap, shake).

* **Vanilla JS Architecture**
  Lightweight and fast, no external dependencies.

---

## 🧠 AI Design Layer

This repository also includes the **Build-Bits UX Claude Skill**, a reusable AI-assisted design module used to define:

* Adaptive difficulty (Flow Zone)
* Robot assistant behavior
* ADHD-focused UX rules
* Game mode logic

The skill can be reused in future projects to guide adaptive and user-centered design.
